# Ex-App2.0 (Under Development)
<h3>
An App that let you explore and Downloads latest , trending Gifs along with creating your own using camera features.
  </h3>
  
  <h2> 
Technoligies Used </h2>

- Firebase
- MVVM
- Retrofit
- Dependency Injection
- Coroutines
- paging3
- glide
- CameraX
- Dagger Hilt
- Live Data
- Nav Graphs
- More to come...
